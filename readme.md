# ピアノ演奏の依頼者と演奏者をマッチングするアプリ
ピアノ演奏を依頼したい団体、個人と演奏者をつなぐアプリ。

## 機能(21/03/27時点)
- ユーザー登録、ログイン機能
- 投稿機能, 画像投稿(S3)
- 編集、削除機能
- チャット機能(Pusher)
　リアルタイム機能についてはローカル環境のみ有効

Getting ready.
